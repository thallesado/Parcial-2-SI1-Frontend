# Public

Archivos estaticos servidos directamente por Next.js.

## Que puede ir aqui

- Imagenes.
- Logos.
- Iconos.
- Archivos que deben estar disponibles por URL publica.

## Subcarpetas

- `imagenes/`: imagenes usadas por el portal publico.

## Ejemplo

Un archivo en:

```text
public/imagenes/logo_uagrm.png
```

se usa en el codigo como:

```text
/imagenes/logo_uagrm.png
```
