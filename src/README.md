# Carpeta src

Contiene el codigo fuente del frontend.

## Subcarpetas

- `app/`: rutas y paginas de Next.js.
- `lib/`: funciones compartidas, cliente API y tipos.

## Regla practica

- Si quieres cambiar una pantalla, entra a `app/`.
- Si quieres cambiar como se llama al backend, entra a `lib/api.ts`.
