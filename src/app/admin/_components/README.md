# Componentes administrativos

Componentes reutilizables dentro del panel administrativo.

## Componentes actuales

- `AdminTable.tsx`: tabla administrativa con estilo uniforme.
- `PaginationControls.tsx`: controles de paginacion.

## Para que sirven

Evitan repetir el mismo diseno de tabla y paginacion en varios modulos.

## Cuando crear otro componente

Crea un componente aqui si:

- se usa en mas de un modulo;
- representa una pieza visual pequena;
- no contiene una regla de negocio completa.
